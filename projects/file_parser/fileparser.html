File Parser

Overview:

This was a two part assignment for SFU's CMPT 365 - Multimedia Systems. We were required to parse a .bmp image to read the file metadata. Then later we were tasked with implementing a lossless compression method to compress the image, display compression details, and then access the compress file to decompress and show the result.

Goal:

  • Display a .bmp image and read the file bits to parse metadata.
  • Modify image by changing data - brightness, scale, color channels.
  • Compress image into custom file format using Huffman encoding and display compression details.
  • Decompress custom file format and display the image losslessly.
